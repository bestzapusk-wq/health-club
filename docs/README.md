# Документация проекта Health Club

## Для Claude Projects

Эти документы предназначены для загрузки в Claude Projects для работы над проектом Health Club.

---

## Структура документации

### 1. PROJECT_SYSTEM_PROMPT.md
**Назначение:** Системный промпт для Claude

Содержит:
- Роль и контекст Claude в проекте
- Описание продукта и его функций
- Текущий этап разработки
- Цели и бизнес-модель
- Как Claude должен помогать
- Важные ограничения

**Использование:** Вставить в System Instructions проекта в Claude

---

### 2. APP_OVERVIEW.md
**Назначение:** Техническое описание приложения

Содержит:
- Архитектура и стек технологий
- Структура страниц и роутинг
- Схема базы данных
- Описание компонентов и сервисов
- Edge Functions
- Статус реализации функций
- Команды разработки

**Использование:** Для технических вопросов и доработок кода

---

### 3. TARGET_AUDIENCE.md
**Назначение:** Описание целевой аудитории

Содержит:
- Портрет идеального клиента
- Проблемы и боли аудитории
- Мотивация и триггеры к покупке
- Возражения и как их преодолевать
- Вторичные сегменты
- Поведение в digital
- Jobs To Be Done

**Использование:** Для маркетинговых задач и продуктовых решений

---

### 4. COMPETITORS_AND_POSITIONING.md
**Назначение:** Анализ конкурентов и позиционирование

Содержит:
- Карта прямых и косвенных конкурентов
- Сравнительные таблицы
- УТП и ключевые отличия
- Конкурентные преимущества
- Барьеры входа и риски
- Стратегия позиционирования
- Ключевые сообщения для маркетинга

**Использование:** Для позиционирования, рекламы, контента

---

### 5. LAUNCH_PLAN.md
**Назначение:** План запуска на рынок СНГ

Содержит:
- Этапы запуска (0-4)
- Чеклисты подготовки
- Стратегии по каналам (Instagram, Telegram, YouTube)
- Бюджет на запуск
- KPI и метрики воронки
- Риски и митигация
- Следующие шаги

**Использование:** Для планирования и отслеживания прогресса

---

### 6. SURVEY_FORMAT.md (существующий)
**Назначение:** Формат данных опросника

Содержит техническое описание структуры вопросов.

---

## Как использовать в Claude Projects

### Шаг 1: Создать проект
1. Открыть claude.ai
2. Нажать "Projects" → "New Project"
3. Назвать "Health Club"

### Шаг 2: Добавить System Instructions
Скопировать содержимое `PROJECT_SYSTEM_PROMPT.md` в поле System Instructions

### Шаг 3: Загрузить документы
Загрузить в Project Knowledge:
- APP_OVERVIEW.md
- TARGET_AUDIENCE.md
- COMPETITORS_AND_POSITIONING.md
- LAUNCH_PLAN.md

### Шаг 4: Начать работу
Теперь в чатах этого проекта Claude будет знать весь контекст и помогать с:
- Продуктовыми решениями
- Техническими вопросами
- Маркетинговыми материалами
- Планированием

---

## Обновление документации

При значительных изменениях в проекте обновляйте соответствующие документы:

- Новые функции → APP_OVERVIEW.md
- Изменение аудитории → TARGET_AUDIENCE.md
- Новые конкуренты → COMPETITORS_AND_POSITIONING.md
- Изменение планов → LAUNCH_PLAN.md
